# Prerequisites for editing
* [Node.js](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) version 18 or higher.
* [pnpm](https://pnpm.io/installation) version 10.17.x.x or higher.
* Terminal for accessing VitePress via its command line interface (CLI).
* Text Editor with Markdown syntax support.
* [VSCode](https://code.visualstudio.com/docs/setup/setup-overview) is recommended, along with the official Vue extension.
After moving to the root directory and installing prereqs:
```sh
$ pnpm install
```
Run the following to start editing:
```sh
$ pnpm run docs:dev
```
